## Проект: Гипотеза №6 — Моделирование сезонности во временных рядах

## Описание
Исследование различных подходов к моделированию сезонности с помощью глобальной модели CatBoost.

## Гипотеза
Комбинация разнородных признаков (лаги, сезонные лаги, календарные признаки, ряды Фурье) улучшает качество прогноза по сравнению с использованием отдельных групп признаков.
